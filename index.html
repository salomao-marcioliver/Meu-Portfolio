Projetos

Projeto 1 - Listas de Tarefas

Descrição: Desenvolvi um sistema web de lista de tarefas utilizando React.js para o frontend e Firebase para o backend. O projeto oferece uma interface dinâmica e responsiva onde os usuários podem adicionar, editar e excluir tarefas facilmente. A integração com Firebase Firestore permite sincronização em tempo real, enquanto o Firebase Authentication garante a segurança e personalização das contas dos usuários. Com uma abordagem mobile-first, o sistema funciona perfeitamente em dispositivos móveis, tablets e desktops, destacando minhas habilidades em desenvolvimento web interativo e escalável.

Projeto 2 - IMC

Descrição: Desenvolvi um sistema web com React.js para calcular o Índice de Massa Corporal (IMC), um indicador crucial da saúde. Os usuários podem inserir seu peso e altura para obter o IMC instantaneamente. Utilizei React.js para criar uma interface interativa e responsiva, proporcionando uma experiência amigável em diferentes dispositivos. O sistema utiliza a fórmula padrão de IMC para resultados precisos e oferece feedback visual sobre o status de peso do usuário. Este projeto destaca minhas habilidades em frontend com React.js, focadas em criar soluções web intuitivas e úteis para a saúde pessoal.

Projeto 3 - Jogo da Memória

Descrição: Desenvolvi o DevMemory, um jogo da memória criado em React.js. Neste projeto, os jogadores são desafiados a encontrar pares de figuras no menor tempo possível. A interface responsiva oferece uma experiência intuitiva em dispositivos diversos. Implementei um cronômetro para registrar o tempo de jogo e um contador de movimentos para acompanhar as tentativas dos jogadores. O jogo não só estimula habilidades de memória e concentração, mas também proporciona feedback visual imediato sobre o progresso. Este projeto destaca minhas habilidades em desenvolvimento frontend com React.js, focado em criar jogos interativos e envolventes.
